### 프로젝트 소개 
Velog를 모티브로 하여 만든 마크다운 에디터 기반의 블로그 웹 애플리케이션 DO! 입니다. 

<br>

### 프로젝트에 사용된 기술
FrontEnd : Vue, Vuex, Vuetify

BackEnd : Spring Boot, AWS EC2, Nginx, Docker

Database : MariaDB

CI/CD : Jenkins

<br>
